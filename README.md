# anti-print

a browser extension that protects your browser fingerprint by blocking canvas-based tracking techniques.

## what is this?

a browser extension that protects your browser fingerprint by blocking canvas-based tracking techniques.

this is a fork of [kkapsner/CanvasBlocker](https://github.com/kkapsner/CanvasBlocker) with a redesigned ui.

## original vs. this version

this is a fork of the [kkapsner/CanvasBlocker](https://github.com/kkapsner/CanvasBlocker) project.

### changes from original

- redesigned the ui with a dark pkm aesthetic. all text is now lowercase. extension renamed from canvasblocker to anti-print. auto-update checking added.

## installation

download the `.xpi` file from releases and open it in firefox, or load it temporarily via `about:debugging` > this firefox > load temporary add-on.
